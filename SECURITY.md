# Security Policy

## Supported Versions

`tapship` is pre-1.0 and currently supported on the latest `main` branch only.

## Reporting a Vulnerability

Please do not open public issues with exploit details.

Open a private report with:

- affected commit or branch
- reproduction steps
- potential impact
- suggested mitigation, if known

Because tapship is local-first and dry-run by default, the most important security risks are accidental credential handling, path traversal, or unexpected network behavior. Reports in those areas are especially helpful.
